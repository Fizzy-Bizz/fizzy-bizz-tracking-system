Soda Shop Sales Analyzer

How to Use This Tool

This application analyzes soda shop sales data from CSV files to help you track drink sales, flavor popularity, and estimate syrup usage.

  1. Upload a CSV file containing your sales data (from your POS system)
  2. Click the "Analyze Sales Data" button to process the file
  3. View the analysis results, including:
    • Total drink sales
    • Sales breakdown by base soda type
    • Sales breakdown by drink size
    • Most popular flavors and recipes
    • Estimated syrup usage for ordering inventory
