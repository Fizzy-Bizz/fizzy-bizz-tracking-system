Fizzy Bizz Sales Analyzer

How to Use This Tool

This application analyzes soda shop sales data from CSV files to help you track drink sales, flavor popularity, and estimate syrup usage.

  1. Upload a CSV file containing your sales data (from your POS system)
  2. Enter your base soda container sizes in gallons (default is 5 gallons if left blank)
  3. Enter the last replacement date for each syrup (leave blank to assume full at current date)
  4. Click the "Analyze Sales Data" button to process the file
  5. View the analysis results, including:
    • Total drink sales
    • Sales breakdown by base soda type
    • Sales breakdown by drink size
    • Most popular flavors and recipes
    • Estimated syrup usage for inventory
    • Syrup replacement forecast showing when each soda syrup will need to be replaced
    • Current syrup levels based on last replacement date
